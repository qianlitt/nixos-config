# mini

## mini.ai

```text
<操作符><范围><对象>
```

|    部分    | 常用键                                           |
| :--------: | ------------------------------------------------ |
| **操作符** | `d` 删除, `y` 复制, `c` 修改, `v` 选中           |
|  **范围**  | `a` = around（含边界）, `i` = inside（不含边界） |
|  **对象**  | `w` 单词, `"` 引号, `f` 函数, `t` 标签...        |

### 内置对象

| 对象            | `i` (inside) | `a` (around) | 示例                         |
| --------------- | ------------ | ------------ | ---------------------------- |
| `"` `"`         | 引号内内容   | 包含引号本身 | `ci"` 修改字符串内容         |
| `'` `'`         | 同上         | 同上         | `da'` 删除整个带引号的字符串 |
| `` ` `` `` ` `` | 反引号       | 反引号       | ``vi` `` 选中反引号内        |
| `( )`           | 括号内       | 含括号       | `di(` 删除括号内参数         |
| `[ ]`           | 中括号内     | 含中括号     | `ya[` 复制含中括号           |
| `{ }`           | 大括号内     | 含大括号     | `ca{` 修改代码块             |
| `< >`           | 尖括号内     | 含尖括号     | `di<<` 删除泛型参数          |

### 自定义对象

| 对象 | 作用                 | 示例操作                                                |
| ---- | -------------------- | ------------------------------------------------------- |
| `o`  | **代码块/条件/循环** | `vio` 选中 if/for 内部，`dao` 删除整个 if 块            |
| `f`  | **函数**             | `vif` 选中函数体，`daf` 删除整个函数                    |
| `c`  | **类**               | `vic` 选中类内部，`yac` 复制整个类                      |
| `t`  | **HTML/XML 标签**    | `vit` 选中 `<div>内容</div>` 的内容，`dat` 删除整个标签 |
| `d`  | **数字**             | `vid` 选中光标下的数字                                  |
| `e`  | **驼峰单词**         | `vie` 选中 `myVariable` 中的 `Variable`                 |
| `g`  | **整个文件**         | `vig` 选中整个 buffer                                   |
| `u`  | **函数调用**         | `viu` 选中 `func(arg)` 的参数，`vau` 选中整个调用       |
