# Hyprland 按键绑定参考

## 应用

| 按键             | 功能         | 说明                         |
| ---------------- | ------------ | ---------------------------- |
| `SUPER + Q`      | 终端         |                              |
| `SUPER + Return` | 终端         | 备用                         |
| `SUPER + Z`      | 快速访问终端 | kitten quick-access-terminal |
| `SUPER + W`      | 浏览器       |                              |
| `SUPER + D`      | 编辑器       |                              |
| `SUPER + A`      | 音乐         |                              |
| `SUPER + E`      | 文件管理器   |                              |

## 窗口管理

| 按键                      | 功能          | 说明                |
| ------------------------- | ------------- | ------------------- |
| `SUPER + 鼠标左键`        | 拖拽窗口      |                     |
| `SUPER + 鼠标中键`        | 拖拽窗口      | 备用                |
| `SUPER + 鼠标右键`        | 调整窗口大小  |                     |
| `SUPER + ←/→/↑/↓`         | 焦点移动      | 方向键风格          |
| `SUPER + H/J/K/L`         | 焦点移动      | Vim 风格（←/↓/↑/→） |
| `SUPER + SHIFT + ←/→/↑/↓` | 移动窗口      | 方向键风格          |
| `SUPER + SHIFT + H/J/K/L` | 移动窗口      | Vim 风格            |
| `SUPER + F`               | 全屏切换      |                     |
| `SUPER + V`               | 浮动/平铺切换 |                     |
| `SUPER + P`               | 固定切换      | 始终置顶            |
| `SUPER + C`               | 关闭窗口      |                     |

## 工作区

### 聚焦工作区

| 按键                   | 功能            | 说明       |
| ---------------------- | --------------- | ---------- |
| `SUPER + 0-9`          | 聚焦工作区 1-10 | 主键盘     |
| `SUPER + 数字键盘 0-9` | 聚焦工作区 1-10 | 数字键盘区 |
| `SUPER + U`            | 左侧工作区      | 相对移动   |
| `SUPER + I`            | 右侧工作区      | 相对移动   |
| `SUPER + Page_↑`       | 左侧工作区      | 相对移动   |
| `SUPER + Page_↓`       | 右侧工作区      | 相对移动   |
| `SUPER + 鼠标滚轮↑`    | 左侧工作区      | 相对移动   |
| `SUPER + 鼠标滚轮↓`    | 右侧工作区      | 相对移动   |

### 移动窗口到工作区

| 按键                         | 功能              | 焦点跟随 |
| ---------------------------- | ----------------- | -------- |
| `SUPER + ALT + 0-9`          | 发送到工作区 1-10 | 否       |
| `SUPER + ALT + 数字键盘 0-9` | 发送到工作区 1-10 | 否       |
| `SUPER + SHIFT + Page_↑`     | 发送到左侧工作区  | 是       |
| `SUPER + SHIFT + Page_↓`     | 发送到右侧工作区  | 是       |
| `SUPER + SHIFT + 鼠标滚轮↑`  | 发送到左侧工作区  | 是       |
| `SUPER + SHIFT + 鼠标滚轮↓`  | 发送到右侧工作区  | 是       |

## 布局

| 按键              | 功能                    | 说明                             |
| ----------------- | ----------------------- | -------------------------------- |
| `SUPER + ALT + [` | 切换布局 ←              | dwindle <-> master <-> scrolling |
| `SUPER + ALT + ]` | 切换布局 →              |                                  |
| `SUPER + ;`       | 减小分割比例            | dwindle 布局，可重复             |
| `SUPER + '`       | 增大分割比例            | dwindle 布局，可重复             |
| `SUPER + [`       | consume_or_expel 上一个 | scrolling 布局                   |
| `SUPER + ]`       | consume_or_expel 下一个 | scrolling 布局                   |
| `SUPER + R`       | 切换列宽                | scrolling 布局                   |
| `SUPER + ,`       | 减小列宽                | scrolling 布局，可重复           |
| `SUPER + .`       | 增大列宽                | scrolling 布局，可重复           |

## Quickshell（noctalia）

| 按键              | 功能       | 说明 |
| ----------------- | ---------- | ---- |
| `SUPER + Space`   | 启动器开关 |      |
| `SUPER + ALT + X` | 控制中心   |      |
| `SUPER + ALT + ,` | 设置面板   |      |
| `SUPER + ALT + L` | 锁屏       |      |

## 媒体

| 按键                    | 功能     | 说明         |
| ----------------------- | -------- | ------------ |
| `XF86AudioRaiseVolume`  | 音量+    | 锁定，可重复 |
| `XF86AudioLowerVolume`  | 音量-    | 锁定，可重复 |
| `XF86AudioMute`         | 静音切换 | 锁定         |
| `XF86MonBrightnessUp`   | 亮度+    | 锁定，可重复 |
| `XF86MonBrightnessDown` | 亮度-    | 锁定，可重复 |
